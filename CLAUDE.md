# CLAUDE.md — AI_Coding

## 项目概述

记录 AI 辅助编程的实战经验，包括 Prompt 技巧、工具使用心得、代码案例和踩坑记录。

## 目录结构

```
AI_Coding/
├── prompts/     # Prompt 模板与技巧
├── tools/       # AI 编程工具使用经验（Claude Code、Cursor 等）
├── cases/       # 实际项目中的 AI 编程案例
└── notes/       # 零散笔记、踩坑记录
```

## 内容规范

- 每个案例/技巧尽量包含：背景、做法、效果
- 踩坑记录要写清楚：现象、原因、解决方法
- 代码示例优先用真实项目片段，脱敏后放入

## 禁止项

- 不提交包含真实密钥、密码、个人信息的文件
- 不提交未脱敏的业务数据

## Git 工作流

- 直接推 main 分支
- commit message 用中文或英文均可，描述清楚即可

工具调用同一操作失败3次后，立即停止重试，不再尝试相同操作。输出：1）错误原因分析；2）可能的解决方案；3）等待用户确认后再继续。

**Why:** 盲目重试浪费时间，且可能造成重复副作用（如重复写入、重复提交）。失败3次说明有根本性问题需要人工介入。

**How to apply:** 任何工具（Write、Edit、Bash、Agent 等）连续失败3次，停止操作，向用户说明原因和解决方案，等确认后再行动。
